
# Unemployment Rate Analysis – Power BI

## Overview
This project presents an interactive Power BI dashboard analyzing unemployment trends across regions and time periods. The dashboard highlights key patterns, comparisons, and insights to support data-driven understanding of employment dynamics.

## Tools & Technologies
- Power BI Desktop
- Power Query
- DAX

## Key Insights
- Year-wise unemployment trends
- Region/state-wise comparison
- Identification of peak and noting periods
- Interactive filters and slicers for exploration

## Dataset
- Source: Public unemployment dataset
- Data cleaning and transformation performed using Power Query

## Files in This Repository
- `unemployment_r.pbix` – Power BI report file
- `dashboard_overview.png` – Dashboard preview
- `trend_analysis.png` – Trend analysis view

## How to View the Dashboard
1. Download the `.pbix` file
2. Open using **Power BI Desktop**
